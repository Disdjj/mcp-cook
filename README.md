# Cook

> 灵感来源: [https://github.com/Anduin2017/HowToCook/issues/1462](https://github.com/Anduin2017/HowToCook/issues/1462)

## 指南

在任意支持 mcp 的 client 中:

```
{
  "mcpServers": {
    "codelf": {
      "command": "npx",
      "args": ["mcp-cook"]
    }
  }
}
```

即可开始使用

## 目前支持两种模式

1. 报菜名
2. 获取食谱
